## 1. 安装与使用

### 1.1 安装方式

进入到工程目录并在 pubspec.yaml 中添加以下依赖：

<!-- tabs:start -->

#### pubspec.yaml

```yaml
...

dependencies:
  vibration: ^3.1.4
  vibration_harmonyos: ^0.0.1
...
```

执行命令

```bash
flutter pub get
```
