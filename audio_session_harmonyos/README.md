## 1. 安装与使用

### 1.1 安装方式

进入到工程目录并在 pubspec.yaml 中添加以下依赖：

<!-- tabs:start -->

#### pubspec.yaml

```yaml
...

dependencies:
  audio_session: ^0.2.2
  audio_session_harmonyos: ^0.0.1
...
```

执行命令

```bash
flutter pub get
```
